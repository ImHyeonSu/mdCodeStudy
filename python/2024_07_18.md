## list の reverse

```python
# メモリ使用率が一番低い(O(1))ただ、元に復元できない。
num_list = [1, 2, 3, 4, 5]
num_list.reverse()
print(num_list) # output: [5, 4, 3, 2, 1]

# メモリ使用率が少し低い
num_list = [1, 2, 3, 4, 5]
num_list = list(reversed(num_list))
print(num_list) # output: [5, 4, 3, 2, 1]

# メモリ使用率がO(n)
num_list = [1, 2, 3, 4, 5]
num_list = num_list[::-1]
print(num_list) # output: [5, 4, 3, 2, 1]
```
