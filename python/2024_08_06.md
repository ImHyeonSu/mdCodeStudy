## split

```python
s = "Hello, my name is John"
segments = s.split()
print(segments)  # output: ['Hello,', 'my', 'name', 'is', 'John']

s = "apple,banana,orange"
segments = s.split(',')
print(segments)  # output: ['apple', 'banana', 'orange']
```

## string to int

```python
binary_string = "101"
decimal_number = int(binary_string, 2)
print(decimal_number)  # output: 5
```
