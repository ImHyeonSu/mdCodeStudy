## leftJoin, innerJoin
JOIN文はFROMの後に作成、ONは条件
``` MySQL
SELECT
FROM
LEFT JOIN or INNER JOIN
ON
```

## 重複など
```SQL
SELECT Email FROM PERSON GROUP BY Email HAVING COUNT(*) > 1;
```

## 日付比較
```SQL
SELECT DATEDIFF('2021-01-16', '2018-03-01');
-> 1

SELECT TIMESTAMPDIFF(QUARTER,'2021-09-22', '2020-03-28');
-> -5
```
